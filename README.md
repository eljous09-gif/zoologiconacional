
## Descripción
Proyecto web desarrollado como plataforma informativa para el Zoológico Nacional. El sitio permite a los visitantes consultar información sobre las especies, servicios, horarios y datos relevantes de contacto, ofreciendo una experiencia de usuario clara y funcional.
## Tecnologías Utilizadas
 * *Lenguaje:* HTML5, CSS3, JavaScript (ES6+).
 * *Diseño:* Responsive Design (adaptable a móviles y escritorio).
 * *Control de versiones:* Git y GitHub.
 * *Despliegue:* GitHub Pages.
## Características Principales
 * *Interfaz intuitiva:* Navegación optimizada para encontrar información de animales y horarios rápidamente.
 * *Diseño Responsivo:* Visualización perfecta en cualquier dispositivo.
 * *Optimización Web:* Estructura limpia para una carga rápida de contenido.
## Instalación
Para ver el código fuente y realizar modificaciones locales:
 1. Clona el repositorio:
   bash
   git clone https://github.com/eljous09-gif/zologiconacional.git
   
 2. Abre el archivo index.html en tu navegador preferido.
 3. (Opcional) Si realizas cambios en el CSS/JS, asegúrate de mantener la estructura de carpetas para que los enlaces funcionen correctamente.
